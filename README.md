# CineVerse – Movie Recommendation App

A Netflix-style movie discovery app built with vanilla HTML, CSS, and JavaScript, powered by the TMDb API.

## Status
Phase 1 – Foundation (project setup complete)

## Structure
- index.html – Home page (navbar, hero, trending/popular sections, footer)
- movie.html – Movie detail page
- css/ – Stylesheets (base, movie detail, responsive breakpoints)
- js/ – Application logic, split by responsibility:
  - app.js – entry point / initialization
  - api.js – TMDb API calls
  - search.js – search + suggestions
  - movie.js – movie detail page logic
  - favorites.js – favorites/watchlist (localStorage)
  - utils.js – shared helper functions
- assets/ – images and icons
